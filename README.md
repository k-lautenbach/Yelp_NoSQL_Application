# Yelp Gentrification Analysis — Indianapolis

## Required Data Structure

The `Yelp-JSON/` folder is not committed. Create it locally with this layout before running `load_mongo.py`:

```
hw6_share/
└── Yelp-JSON/
    └── filtered_indianapolis/
        ├── yelp_academic_dataset_business.json
        ├── yelp_academic_dataset_review.json
        ├── yelp_academic_dataset_checkin.json
        └── yelp_academic_dataset_tip.json
```

These are Indianapolis-filtered subsets of the [Yelp Open Dataset](https://www.yelp.com/dataset).
