# Yelp Gentrification Analysis — Indianapolis

Using the [Yelp Open Dataset](https://www.yelp.com/dataset) and MongoDB to quantify gentrification in **Indianapolis** by analyzing:

- Business category shifts by neighborhood over time
- Price tier changes (Yelp `attributes.RestaurantsPriceRange2`)
- Review velocity by neighborhood over time

## Setup

### 1. Download the Yelp Academic Dataset

1. Go to <https://www.yelp.com/dataset/download> and agree to the Dataset License.
2. Download the JSON package (`yelp_dataset.tar` or `yelp_academic_dataset.zip`).
3. Extract the contents so the JSON files land in:
   ```
   Yelp-JSON/Yelp JSON/
   ```
   You should end up with the five `.json` files listed in the structure above.

> The raw dataset is ~4 GB and is excluded from this repo via `.gitignore`. Do **not** commit the JSON files.

### 2. Filter to Indianapolis

Run the filter script once to generate the `filtered_indianapolis/` subset:

```bash
python read.py
```

This reads from `Yelp-JSON/Yelp JSON/` and writes Indianapolis-only records to `Yelp-JSON/filtered_indianapolis/`.

### 3. Load into MongoDB

*(Analysis scripts coming soon.)*

Start a local `mongod` instance, then import the filtered files:

```bash
mongoimport --db yelp_indy --collection businesses \
  --file "Yelp-JSON/filtered_indianapolis/yelp_academic_dataset_business.json"

mongoimport --db yelp_indy --collection reviews \
  --file "Yelp-JSON/filtered_indianapolis/yelp_academic_dataset_review.json"

mongoimport --db yelp_indy --collection checkins \
  --file "Yelp-JSON/filtered_indianapolis/yelp_academic_dataset_checkin.json"

mongoimport --db yelp_indy --collection tips \
  --file "Yelp-JSON/filtered_indianapolis/yelp_academic_dataset_tip.json"
```

## Requirements

- No additional Python packages required for `read.py` (stdlib only)
